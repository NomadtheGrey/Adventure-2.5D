# PLAYER_GUIDE.md: SURVEYOR MISSION BRIEFING

## 1. THE MISSION: "REBOOT THE SECTOR"
The Labyrinth is a corrupted memory sector. To stabilize it, you must retrieve the **Source Code (The Chalice)** and return it to the **Root Directory (Gold Throne)** in the north.

## 2. THE CONTROLS (P.A.S. INTERFACE)
*   **[WASD]**: Navigational thrust.
*   **[SPACE] / [CLICK]**: **Pulse Spear Thrust.** Use this to disrupt the logic of Security Daemons (Dragons).
*   **[Q]**: **Drop Item.** You can only carry one high-level artifact at a time.
*   **[SCROLL / 1-9]**: Cycle active modules (Magnet, Bridge, Spear).
*   **[ESC]**: Pause / Access System Diagnostics.

## 3. SECURITY DAEMONS (DRAGONS)
The sector is patrolled by three classes of Daemons:
1.  **RHYNODON (Red):** A relentless hunter. If it detects your signal, it will never stop tracking you.
2.  **GORGARYS (Green):** A sentinel guarding keys. It is passive until its charge is stolen.
3.  **YVITHRAX (Yellow):** A baiter. It will steal your dropped items and move them to random sector coordinates.

## 4. CRITICAL UTILITIES
*   **The Pulse Spear:** Your only defense. Hit a Daemon in the "Head" (core node) to delete its process.
*   **The Flux Attractor:** Uses magnetism to pull nearby artifacts toward you.
*   **The Phase Bridge:** Allows you to warp through dense nature (Trees/Bushes) or across water.

## 5. SIGNAL INTEGRITY
Your health is measured in **Signal Integrity**. Contact with a Daemon causes massive packet loss. If Integrity reaches 0%, the link is terminated and the sector resets.
